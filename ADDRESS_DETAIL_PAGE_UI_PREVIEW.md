## Key UI Elements

1. **Back Button**: Ghost button with arrow icon, top-left
2. **Address Display**: Monospace font in muted background, with copy button
3. **Statistics Cards**: Elevated cards with icon, label, and value
4. **Search Bar**: Full-width search input with icon
5. **View Toggle**: Icon buttons for grid/list view
6. **Filters**: Sidebar with radio buttons and checkboxes
7. **Asset Grid**: Responsive grid of AssetCard components
